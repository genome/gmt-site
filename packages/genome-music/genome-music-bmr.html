sub commands

calc-covg Uses calcRoiCovg.c to count covered bases per-gene for each given tumor-normal pair of BAMs.
calc-bmr Calculates mutation rates given per-gene coverage (from "music bmr calc-covg"), and a mutation list
calc-covg-helper Uses calcRoiCovg.c to count covered bases per-gene for a tumor-normal pair of BAMs.
calc-wig-covg Count covered bases per-gene for each given wiggle track format file.
