Frequently Asked Questions

I have a question that isn't addressed here. How can I get help?

We encourage you to search for similar questions on BioStar or, if needed, to ask a new question there. By participating in the BioStar community, you will benefit from broader support while others will benefit from your question. We automatically monitor BioStar for questions related to Genome Modeling Tools (and other topics of interest) to ensure that your questions are answered. We hope to see you there!

What Linux distributions do you support?

Ubuntu Lucid 10.04 64-bit.

Can I install Genome Modeling Tools on other operating systems (e.g., RedHat Linux, OS X)?

We recommend that users of other operating systems download the Ubuntu "64-bit PC (AMD64) desktop CD" image listed on the Ubuntu 10.04 downloads page and install it in a virtual machine (such as VirtualBox). Then, install the Genome Modeling Tools on that VM.

When will Ubuntu 12.04 packages be released?

Creating Ubuntu 12.04 packages is a priority for us but we cannot yet commit to a timeline. We want to ensure we can release all of tools simultaneously for any supported platform. More importantly we want to ensure we can offer timely updates to all supported platforms. As such, we only support Ubuntu Lucid 10.04 64-bit at this time.
